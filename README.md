# NFC-UNIFIED

Этот каталог создан для безопасного сращивания двух проектов без изменения исходников.

## Что уже сделано

- Источник 1 (админка Django) скопирован в `sources/site_admin`.
- Источник 2 (редактор/preview Flask) скопирован в `sources/hexgraph_editor`.
- Исключены тяжелые/локальные артефакты: `venv`, `__pycache__`, базы данных, node/playwright мусор.

## Цель объединения

Собрать единый проект, где:

1. Админка генерирует NFC-ссылки и показывает статистику.
2. По ссылке `/<token>`:
   - до публикации открывается редактор (из Hexgraph),
   - после публикации открывается просмотр открытки.
3. Публикация выполняется только после preview + подтверждения.

## Архитектура (рекомендованная)

Базой оставить Django-проект (`site_admin`) и встроить UI/flow редактора из Hexgraph.

- `Card` из Django остается основной сущностью ссылки и статуса публикации.
- HTML/CSS/JS редактора переносится в Django templates/static.
- Маршруты preview/finalize адаптируются под `Card`.

Подробный план: `unified/MERGE_PLAN.md`.

## Старт объединения

1. Создать ветку в этом каталоге.
2. Копировать шаблоны редактора в Django app (новые routes: edit/preview/finalize).
3. Подключить сохранение в модель `Card`.
4. Проверить полный flow NFC.
